# 📝 Notes App (React)

A simple and interactive **Notes App** built using **React.js**.
This project allows users to create and delete notes with a clean UI and responsive layout.

## ✨ Features

* ➕ Add new notes (title + description)
* 🗑️ Delete notes instantly
* 🔄 Real-time UI update using React state
* 📱 Responsive layout (mobile + desktop)
* ⚡ Fast performance with React

---

## 🛠️ Tech Stack

* **React.js** – Frontend library
* **JavaScript (ES6+)** – Logic
* **Tailwind CSS** – Styling
---

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/ayushh025/Notes-app.git
```

2. Navigate to the project folder:

```bash
cd Notes-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
